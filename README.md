# ReflectionSupport 反射工具箱

# 版本 : 2.0
# 状态
开发中

# 功能介绍
1. 封装了常用的反射操作
2. 提供了常用的处理方案

支持最小的JDK版本:JDK 1.8

## 介绍
包含大量常用的反射或普通方法的封装，可以方便开发的过程和流程

# 本版本不和1.X版本兼容！ 请注意

# 目前以支持功能：
对任意对象完全反射支持
# 将被添加的功能
1. 随机生成支持

    1. 扩展随机生成的局限性
    2. 支持定制随机生成
    3. 任意类型的随机生成
    4. 列表等集合类单独处理
    5. 数组定制
    6. 支持快速测试
    
2. 字符串转换支持
    1. 扩展字符串的功能
    2. 对目标要求格式进行转换
