```xml

<dependency>
    <groupId>io.github.mxudong</groupId>
    <artifactId>ReflectionSupport</artifactId>
    <version>2.0</version>
</dependency>

```

# 目前以支持功能： 对任意对象完全反射支持
    
    支持数据转换：对象转map， map转对象 
    支持部分方法反射调用
    支持数据随机化
