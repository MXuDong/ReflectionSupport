# ReflectionSupport 依赖支持（安装）

    * 为版本号！请自行替换

# Maven
```xml
<!-- https://mvnrepository.com/artifact/io.github.mxudong/ReflectionSupport -->
<dependency>
    <groupId>io.github.mxudong</groupId>
    <artifactId>ReflectionSupport</artifactId>
    <version>*</version>
</dependency>

```
复制到Maven工程的pom文件

# Gradle
```groovy
// https://mvnrepository.com/artifact/io.github.mxudong/ReflectionSupport
compile group: 'io.github.mxudong', name: 'ReflectionSupport', version: '*'
```
